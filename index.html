游戏规则 · 快速入门

基础概念

每位玩家有生命值、手牌、状态区与装备区。卡牌分为：法术 / 状态 / 武器 / 宝具（阶数：一阶 / 二阶 / 三阶）。回合内可打出卡牌、发动效果、攻击对手等。

回合流程（简化）

  1. 回合开始：占卜/判定效果结算
  2. 摸牌阶段：按规则抽取卡牌
  3. 出牌阶段：可打出法术/装备/使用状态/发动技能
  4. 攻击阶段：使用武器发起攻击
  5. 回合结束：结算状态、技能等

特殊关键词

余烬：卡牌在送入墓地后拥有额外效果。
引渡：放弃升阶从牌河定向抽牌。
运势：投掷骰子，根据点数获得额外效果，运气决定一切...坏运气也是。
占卜：观看牌顶卡牌并任意调换顺序，局势尽在掌握。
共鸣：特定条件触发效果，随机万变。
